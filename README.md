# Rental calculator

Minimal rental cost calculator with real-time currency conversion.

## Features

- Price per day in 5 currencies (USD, EUR, AMD, RUB, AED)
- Include/exclude last day toggle
- Live exchange rate conversion
- Dark mode
- Geist font, Apple/shadcn hybrid UI

## Stack

Vite + React 19 + TypeScript + Tailwind CSS v4.

## Develop

```bash
npm install
npm run dev
```
